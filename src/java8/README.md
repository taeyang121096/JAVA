# JAVA8


### 1. Lambda expression (람다 표현식)
    람다란 이름이 없는 함수를 말합니다. (익명함수)라고 할 수 있습니다.
    기본적인 표현식의 경우 (parameters) -> expression
```java
    1. () -> new Object()
    2. (String s) -> s.length()
    3. (int a, int b) -> a*b
```
